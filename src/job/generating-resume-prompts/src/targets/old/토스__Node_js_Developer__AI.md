### AI

Node.js Developer 토스 소속 정규직 합류하게 될 팀에 대해 알려드려요

Node.js Developer는 웹 자동화 기술과 AI Agent 기반 워크플로우를 통해, 토스 내부의 업무 자동화와 개발 생산성 도구를
만드는 팀에 합류하게 돼요. Web Automation Platform 팀에 소속되어, AI-DX 팀과 긴밀하게 협업하며 AI 응용 도구와 에이전트
기반 자동화 시스템을 함께 설계하고 개발해요. MCP 서버, 코드리뷰 자동화, 플러그인 마켓플레이스 등 토스 전체 개발자의
생산성을 높이는 인프라를 직접 만들고 운영해요. 새로운 시스템을 구축할 때, 많은 팀 구성원이 모여 코드 리뷰를 해요. 버그나
네이밍 미스, 컴포넌트 설계에 어색한 부분은 없는지 함께 리뷰하고 코드의 질적 향상에 대해 끊임없이 토론해요. 일주일에
한번씩 엔지니어링 데이를 해요. 다양한 계열사에서 얻은 기술적인 지식들을 공유받고, 공유할 수 있어요. 합류하면 함께할
업무예요

AI Agent 워크플로우를 설계하고 개발해요. LLM을 활용한 자동화 파이프라인을 구축하여 사내 반복 업무를 근본적으로 해결해요.
MCP(Model Context Protocol) 서버를 개발하고 운영해요. AI 에이전트가 사내 시스템과 안전하게 상호작용할 수 있는
인터페이스를 만들어요. 자금팀, HR, 세일즈 등 다양한 조직의 업무 자동화를 직접 실행해요. 요구사항을 분석하고 최적의
자동화 솔루션을 설계해요. 사용자의 모바일 금융 활동이 더 쉽고 자연스러워질 수 있도록, 다양한 금융 데이터를 자동으로
수집하고 처리하는 시스템을 개발해요. 공용 라이브러리, CLI 도구, 플러그인 등 개발자 생산성 도구를 만들고 개선해요. OOP와
FP 기반의 높은 코드 품질과 충분한 테스트 커버리지를 갖춘 프로젝트를 운영하며, 기능 추가와 유지보수가 쉬운 코드를
지향해요.

이런 분과 함께하고 싶어요

AI Agent나 LLM 기반 애플리케이션을 설계하거나 개발해 본 경험이 있으신 분을 찾고 있어요. 반복적인 업무 프로세스를
분석하고 자동화한 경험이 있으신 분과 함께하고 싶어요. 고객에게 안정적으로 데이터를 제공하고, 외부 상황에 따라 서비스가
중단되지 않도록 모니터링해 본 경험이 있으신 분을 찾고 있어요. 유연하면서도 가독성이 좋은 코드, 유지보수가 가능한 코드를
작성하실 수 있는 분을 찾고 있어요. 이력서는 이렇게 작성하시는 걸 추천해요

AI Agent, LLM API, 프롬프트 엔지니어링 등 AI 응용 개발 경험이 있다면 기재해주세요. 불편한 인터페이스나 반복 업무를
자동화한 경험을 작성해 주세요. Node.js, ELK, ORM 등 다양한 기술을 활용한 웹/앱 서비스 운영 경험을 작성해 주세요. AWS,
GitHub, GOCD 등의 경험이 있으시면 도움이 돼요. 관련 개발 경험이 있으시다면 기재해주세요. Functional Programming, Object
Oriented Programming, Test Driven Development 경험이 있으시다면 기재해주세요. 토스에서 사용하는 기술

TypeScript, Node.js, Nest.js, Go, ELK mikroORM, MySQL, Redis, Kafka LLM API (Claude, GPT), MCP (Model Context Protocol)
