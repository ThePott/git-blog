- 반복되는 정보는 자동 완성으로 입력

## 후보 기술 스택

- AG Grid
    - 다중 선택이 유료 기능으로 들어감
    - formula가 유료 기능으로 들어감
    - 유료 플랜 없이는 원하는 스프레드 시트를 만들 수 없지만, 가격이 부담스러움
- jspreadsheet ce
    - 무료
    - react library가 아니라 native javascript library이기 때문에 리액트 컴포넌트를 셀 안에 넣을 수가 없음
    - 셀 안에 컴포넌트를 만들 땐 선언적으로 못 하고 명령적으로 생성, 삭제를 해야 함
- TanStack Table
    - 무료
    - 테이블 내의 특정 셀의 행, 열, 값에 접근할 수 있음
- just table tag
    - 어차피 내가 다 만들어야 하면 그냥 처음부터 만들면 되지 않나

### 선택

- 전부 처음부터 만들어도 되겠다 생각해서 네 번째 선택지 고름
    - 그러나 virtual scroll 적용 과정에서 문제 발생
        - position absolute가 되다보니 header와 body 간의 연결이 끊김
        - body cell의 너비에 맡게 header cell 너비를 맞춰야 함
        - table 전체에서 각 cell의 너비를 감지하는 기능을 만들어야.
            - 이건 TanStack Table에서 이미 구현해놓은 것
            - 그렇다면 내가 구현하기보단 만들어진 걸 쓰자
