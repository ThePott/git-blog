# 정규표현식에서 앞부분이 너무 많이 차지하는 문제

- 발생 시각: 2026-01-06 14:50
- 문제: 정규표현식에서 `+`때문에 뒷부분의 `+`가 먹힘
- 해결 시각: 2026-01-06 15:02

## 배경

- "문제 123~456"을 "문제 ", 123, 456으로 분리해야 함
- 기존 정규 표현식: /(.+)?(/d+)~(/d+)$/
- "문제 123~456" -> "문제 12", 3, 456

## 분석

- (.+)?가 최대한으로 잡아먹음

### 해결책

- /^(.\*?)(\d+)~(\d+)$/
- ^: start
- \*: 0 or more
- inside ?: non-greedy (as little as possible)
