## shebang (or hashbang)

- ex) `#!/usr/bin/env bash`
- tells how to run this script
- must be placed at the top of the script file
- `#!`: this is shebang line
- `/usr/bin/env`: use this file
- `bash` in bash mode

## chmode: change mode

- x: executable
- -x: make it NOT executable
- +x: make it executable
