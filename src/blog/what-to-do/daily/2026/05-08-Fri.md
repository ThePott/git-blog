- [x] ctrl c 로 끝내기
- [x] word bank 만들기 (두 단어)
- [x] 화면에 work bank 단어 모두 띄우기
- [x] 유저 인풋 결과를 밑에 출력하기
- [x] 카운트를 맨 밑에 보여주기

- [x] word bank text 넘어서는 오타도 화면에 띄우기
- [x] 완료 로직 찾기
- [ ] 유저 인풋이 단어랑 같으면 DONE! 띄우기
- [ ] DONE 뜨면 통계 보여주기
